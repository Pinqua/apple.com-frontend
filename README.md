## Run Locally

Clone the project

```bash
  git clone https://github.com/Pinqua/apple.com-frontend.git
```

Go to the project directory

```bash
  cd apple.com-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
